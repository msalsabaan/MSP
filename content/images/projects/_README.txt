PROJECT IMAGES
==============

Drop project photos here. The file name you type in the "Cover Image File"
and "Gallery Image Files" columns of the Projects tab MUST match a file here.

Naming rule (lowercase, hyphens, project number based on the row):
  Cover image:    project-01-cover.jpg
  Gallery images: project-01-1.jpg, project-01-2.jpg, project-01-3.jpg ...

Tips:
- Use JPG for photos, PNG only if transparency is needed.
- Cover image: landscape, at least 1600 px wide.
- Keep each file under ~2 MB if possible.
- In the Excel "Gallery Image Files" cell, separate names with a semicolon:
    project-01-1.jpg; project-01-2.jpg; project-01-3.jpg
